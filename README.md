Trello - Mini Task Manager

Это мой первый Java Spring проект, где я реализовал простой менеджер задач.
Проект создан с использованием Java, Spring Boot, Thymeleaf, HTML/CSS, Gradle.

## Функционал:
- Создание папок
- Создание категорий
- Создание задач
- Просмотр деталей(задач,папок,категорий)
  
 Возможность редактировать и удалять элементы.
 
Полностью рабочий интерфейс на страницах .html

## Технологий:
- Java 17
- Spring Boot
- Thymleaf
- HTML/CSS
- Gradle

## Запуск проекта:
1) Склонировать репозиторий
2) Открыть проект в IntelliJ IDEA
3) Дождаться загрузки зависимостей Gradle
4) Запустить TrelloApplication.java
5) Перейти на http://localhost:8020
   
 Планируется улучшение UI и добавление новых функций
